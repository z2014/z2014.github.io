前端优化法则

原来总是迷恋于前端各种效果的实现,但现在网页中,
大型的电商网站不需要那么多炫酷的效果，而前端的性能优化就显得尤为重要。

看了这篇Yahoo的前端优化性能规则之后,

其中有一些还不太懂,先记录一下别人的经验.

内容

* 减少http请求
* 减少DNS查询
* 避免重定向
* Ajax缓存化
* 后载入组件
* 预先载入组件
* 减少DOM元素数量
* 跨域分离组件
* 最少化iframe
* 避免404错误

服务端

* 使用CDN
* 使用过期时间或者可控制缓存的请求头
* Ajax请求使用Get方法
* 避免空的图片src

css

* 将样式放在顶部
* 避免使用css表达式
* 选择而不要@import

javascript

* 将js放在底部
* 使用外部的js

图片

* 优化图片
* 在html中不要缩放图片
* 制作小的和可缓存的favicon.ico
